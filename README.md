# Iglesia Bautista Del Sur

Bilingual Spanish-first starter website for Iglesia Bautista Del Sur.

## Local Development

```powershell
npm.cmd install
npm.cmd run dev
```

## Production Build

```powershell
npm.cmd run build
```

## Vercel Settings

- Framework Preset: Vite
- Root Directory: `./`
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: `dist`

## Language Behavior

The site defaults to Spanish, automatically switches to English for English-language browsers, and lets visitors choose Spanish or English with the ES/EN toggle.
